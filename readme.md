# paddock

A random F1 quote or meme terminal ASCII art generator.

## requirement

- python

## installation

```bash
pipx install git+https://github.com/shadowash8/paddock.git
```

## development

```bash
git clone https://github.com/shadowash8/paddock.git
cd paddock

python -m venv .venv
source .venv/bin/activate

pip install .
```
